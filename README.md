# Password Generator

A clean, minimal password generator that creates secure random passwords instantly. Built with vanilla HTML, CSS and JavaScript.

## Features

- Generates 4 unique random passwords at once
- Customisable password length (8–25 characters)
- Passwords include uppercase, lowercase, numbers and special characters
- Click any password to copy it to your clipboard
- Visual copy confirmation — no annoying alert popups

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Ionicons

## How to Run Locally

```bash
# Clone the repo
git clone https://github.com/Bindekere/Password-generator.git

# Open index.html in your browser
```

No dependencies or build step needed — just open `index.html`.
